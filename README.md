# Develpment Process

```sh
# dev mode on develop on the browser
pnpm dev

# build js
pnpm build

# sync native plugin
pnpm sync:android
pnpm sync:ios

# copy build to native
pnpm copy:android
pnpm copy:ios

# run native
pnpm run:android
pnpm run:ios

# build native
pnpm build:android
pnpm build:ios
```
